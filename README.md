Project Title:
House Market & Sales Performance Analytics Dashboard

Short Description:
This project delivers a multi-page Power BI dashboard to analyze the housing market by region, sales type, and house type, providing insights into prices, sales growth, offer vs purchase trends, and key market influencers.

Tech Stack:
* Power BI
* DAX
* Microsoft Excel / CSV
* SQL
  
Data Source:
Historical housing market dataset containing sales prices, offer prices, regions, cities, house types, sales types, dates, inflation, interest rates, and yield metrics.

Features:
* House market overview with median price change by region
* Sales performance analysis by region and sales type
* Offer price vs purchase price comparison
* YOY sales growth analysis
* Average SQM and SQM price by house type
* Inflation, interest, and yield analysis by house type
* Key influencer insights for price changes
* Interactive slicers for area, city, region, and sales type
* Clean, multi-page dashboard design suitable for business insights
